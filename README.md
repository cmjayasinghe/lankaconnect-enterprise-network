# LankaConnect Enterprise Network

## Overview

This project demonstrates the design and implementation of a secure enterprise campus network using Cisco Packet Tracer.

The project simulates a real-world enterprise environment with multiple departments, enterprise routing, switching, VoIP, wireless networking, firewall security, DMZ, and Layer 2 security features.
