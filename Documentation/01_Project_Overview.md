# Project Overview

## Project Title

**Enterprise Campus Network Design and Secure Implementation for LankaConnect Telecommunications (Pvt) Ltd using Cisco Packet Tracer**

---

## Introduction

LankaConnect Telecommunications (Pvt) Ltd is a fictional Sri Lankan telecommunications and IT solutions provider headquartered in Colombo. The company offers enterprise networking, cloud computing, managed IT services, cybersecurity solutions, and VoIP services to organizations across Sri Lanka.

As the company continues to expand, it requires a modern, secure, scalable, and highly available network infrastructure that supports business operations while protecting critical data and services.

This project demonstrates the design and implementation of a complete enterprise campus network using Cisco Packet Tracer. The solution follows Cisco enterprise networking best practices and incorporates routing, switching, wireless networking, VoIP, firewall security, and network management technologies.

---

# Project Objectives

The primary objectives of this project are to:

* Design a hierarchical enterprise campus network.
* Implement secure network segmentation using VLANs.
* Configure inter-VLAN routing using multilayer switches.
* Provide dynamic IP address allocation through DHCP.
* Implement secure routing using OSPF.
* Deploy secure remote administration using SSH.
* Implement enterprise wireless networking using a Wireless LAN Controller (WLC) and Lightweight Access Points.
* Deploy VoIP services for internal communication.
* Protect the network using a Cisco ASA Firewall with NAT, security zones, and access control policies.
* Provide connectivity to Microsoft Azure cloud resources.
* Ensure the network is scalable, reliable, and easy to manage.

---

# Project Scope

The project includes the complete design and implementation of an enterprise network for a two-floor office building containing multiple business departments.

The implementation includes:

* Enterprise network topology

* OSPF dynamic routing
* DHCP services
